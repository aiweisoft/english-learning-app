# 轻松学英语

基于 uni-app (Vue 3) 的英语单词学习应用，纯前端，无后端依赖。

## 功能

- **每日一句** — 100 条英文名言，按日期固定显示，可前后翻阅
- **分类学习** — 6 个内置分类（小学/初中/高中/日常口语/商务英语/餐饮食品），共 8941 词
- **分组学习** — 每 10 词一组，逐组解锁，进度持久化
- **单词闪卡** — 3D 翻转卡片，点击读发音
- **选择题测验** — 看英文选中文，支持随机/分组出题
- **拼写测试** — 看中文写英文，实时判定对错
- **学习统计** — 已学词数、打卡天数、历史成绩
- **词库管理** — 支持导入/导出词库、按分类增删单词、新增自定义分类

## 词库管理

### 整库操作
- **下载模板** — 获取固定格式的导入模板文件
- **导入词库** — 粘贴 pipe-delimited（`|`）格式词表，替换当前词库
- **导出词库** — 将当前词库导出为文本文件，方便分享和备份
- **恢复内置** — 一键恢复原始 6 分类 8941 词内置词库

### 分类级操作
- **新增分类** — 创建自定义分类（如"医学英语"、"雅思词汇"）
- **删除分类** — 删除空分类（至少保留 1 个）
- **管理单词** — 进入单分类单词列表，支持：
  - 逐条添加单词（英文 + 中文 + 词性）
  - 勾选后批量删除单词
  - 导入词表到该分类（可选择追加合并或替换覆盖）
  - 导出该分类所有单词

### 导入格式

```
# 注释行以 # 开头，空行跳过
英文单词|中文释义|词性(可选)|分类名称(可选)

apple|苹果|n|水果
beautiful|美丽的|a|
run|跑步;运行|v|
```

- 词性代码：`n`=名词、`v`=动词、`a`/`adj`=形容词、`adv`=副词 等
- 未指定分类的单词归入"自定义词库"
- 音标和例句自动生成
- 内置词库首次编辑时自动复制到本地存储，原始文件不受影响

## 技术栈

| 层 | 技术 |
|---|------|
| 框架 | uni-app (Vue 3 + Options API) |
| 样式 | 纯 CSS、rpx 单位、Material 暖色主题 |
| 存储 | `uni.storage` (localStorage) |
| 拼读 | Web Speech API (H5) / 有道 TTS (App) |
| 构建 | HBuilderX 内置 Vite |

## 快速开始

1. 用 [HBuilderX](https://www.dcloud.io/hbuilderx.html) 打开项目
2. 点击"运行" → 选择目标平台（H5 / 微信小程序 / App）
3. H5 指定启动页：`?page=pages/quiz/quiz`

## 项目结构

```
English/
├── App.vue                          # 根组件 + 全局样式
├── pages.json                       # 路由：4 Tab + 6 子页面
├── data/
│   ├── words.js                     # 内置词库（8941 词、100 条名言）
│   ├── custom-words.js              # 自定义词库存储层（导入/导出/增删改）
│   ├── word-service.js              # 统一数据入口（自动切换内置/自定义）
│   └── generate-words.js            # 词库生成器脚本
├── pages/
│   ├── index/                       # 首页（每日一句、分类网格）
│   ├── vocabulary/                  # 单词本（列表 + 闪卡）
│   ├── practice/                    # 练习入口（选择 + 拼写）
│   ├── quiz/                        # 选择题测验
│   ├── spell/                       # 拼写测试
│   ├── word-detail/                 # 分组学习（10 词/组）
│   ├── result/                      # 结果页
│   ├── profile/                     # 我的（统计、打卡、词库管理入口）
│   ├── word-manage/                 # 词库管理（整库导入/导出、分类列表）
│   └── word-list/                   # 分类单词管理（增删、导入/导出单分类）
└── static/tab/                      # 底栏图标（PNG + SVG）
```
